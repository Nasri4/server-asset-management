# Server Asset Management System

Enterprise-level IT Asset Management platform built with:

Backend:
- Node.js
- Express.js
- Microsoft SQL Server
- JWT Authentication
- Role-Based Access Control (RBAC)

Frontend:
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Axios API Layer

---

## Features

- Secure Authentication (JWT)
- Role & Permission System
- Team-based Access Control
- Server Hardware Tracking
- Network & Security Modules
- Dashboard Analytics
- Premium UI Design

---

## Project Structure

- /backend → API server
- /frontend → Next.js application

---

## Getting Started

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev

---

## Environment Variables (Backend)

Create a .env file inside backend:

PORT=5000
DB_USER=your_user
DB_PASSWORD=your_password
DB_SERVER=localhost
DB_DATABASE=SERVER_ASSET_MANAGEMENT
JWT_SECRET=your_secret_key
